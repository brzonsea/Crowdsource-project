Welcome to Map it!

You can use this platform to create a summary in form of a structure map together with your friends.

What is a structure map?

A structure map is similar to a mind-map but it additionally enables you to add relationships between different concepts to the map. These relationships show the influence that these concepts have on each other, the same way as for example quality can influence the cost of a system.

How do I edit the map?

Just click on a node to edit it's details in the sidebar.If you double-click a node, you can see it's details as a pop up. For relationships it works the same way.

Anything else?

Yeah. This is a prototype. Even though we did our best to implement everything correctly, there are probably still some errors. So if you find some please tell us and don't abuse them. Also please provide us with feedback after checking out the platform. We would appreciate it, if you tell us what you liked, what we need to improve, what features you missed or if there's anything else we should know.
